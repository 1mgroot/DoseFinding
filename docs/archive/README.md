# Archive

这里保存的是历史文档、阶段性总结和已经被新版文档覆盖的解释材料。

这些文件没有删除，是为了保留开发过程和旧判断。但它们不再作为主要入口，里面可能包含过时状态，例如“项目 100% 完成”、旧的 PoC 校准目标、旧路径，或已经被新版文档合并的说明。

当前主要入口请看 `docs/README.md`。

## 文件说明

- `PROJECT_OVERVIEW.md`：早期项目总览，内容较完整，但部分状态判断偏乐观。
- `PROJECT_STATUS_FINAL.md`：早期“最终状态”总结，和当前重新审查后的状态不完全一致。
- `PROJECT_ANALYSIS_THREE_QUESTIONS.md`：三问题分析稿，核心内容已合并进 `CHATGPT_PROJECT_CONTEXT.md` 和朗读稿。
- `STATUS_AS_BUILT.md`：旧的 code-first 状态说明，主要内容已由 `STAT_METHODS_AS_BUILT.md` 和新上下文文档覆盖。
- `NEXT_STEP_PLAN.md`：PoC 校准与 flat null 场景的实施计划，保留作历史记录。
- `CALIBRATION_FIX_SUMMARY.md`：校准问题根因分析与修复记录。
- `CALIBRATION_IMPLEMENTATION_SUMMARY.md`：校准系统实施总结。
- `PROJECT_GUIDE.md`：旧的项目指南，和 README/HOW_TO_RUN 重复。
- `QUICK_START.md`：旧的快速开始文档，和 README/HOW_TO_RUN 重复。
- `generated/STAT_METHODS_AS_BUILT.html`：`STAT_METHODS_AS_BUILT.md` 的旧 HTML 渲染版本。
